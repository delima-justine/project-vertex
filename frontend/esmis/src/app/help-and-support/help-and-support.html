The Supply Management Information System (SMIS) is a comprehensive platform designed to streamline inventory management, supply requests, and distribution processes within the organization.

Key features include:

  • Centralized inventory tracking
  • Automated request processing
  • Real-time status updates
  • Comprehensive reporting tools

The system is accessible to:

  • System Administrator: Manages the overall system, users, and configurations
  • Inventory Managers: Handle inventory tracking and supply distribution
  • Department Heads: Review and approve supply requests
  • Authorized Personnel: Staff with specific permissions based on their role

Each user type has different permissions and access levels tailored to their responsibilities.

SMIS is a web-based application that works on most modern devices. Recommended requirements:

  • Browser: Chrome (latest), Firefox (latest), Edge (latest)
  • Internet Connection: Stable broadband connection
  • Screen Resolution: Minimum 1366 x 768

For optimal performance, ensure your browser is updated to the latest version.

To manage user accounts:

  1. Navigate to "User Management" in the sidebar
  2. View all registered users and their status
  3. Use the "Add User" button to create new accounts
  4. Click on any user to edit their information or change permissions
  5. Use the activate/deactivate options to control access

Remember to assign appropriate roles based on user responsibilities.

To reset your password:

  1. Go to "Forget Password"
  2. Put your email
  3. Click "send reset link"
  4. Check your email
  5. Click the link to reset your password.

Note: You have 10 mins to do so if fail to change, request another link to reset.

To review and approve supply requests:

  1. Navigate to "Orders" → "Pending" to see all pending requests
  2. Click on a request to view detailed information
  3. Review the requested items, quantities, and justification
  4. Check inventory availability
  5. Click "Approve" or "Disapprove" with appropriate comments

Approved requests will automatically move to the "Approved" section for processing.

To manage inventory:

  1. Access the inventory management section
  2. Monitor stock levels and set reorder points
  3. Add new items to the inventory catalog
  4. Update quantities when new supplies arrive
  5. Generate reports on inventory usage and trends

Regular monitoring helps prevent stockouts and ensures smooth operations.

To generate reports:

  • Navigate to the "Reports" section in the sidebar
  • Select the type of report you need (inventory, requests, usage, etc.)
  • Set the date range and filter criteria
  • Choose export format (PDF, Excel, etc.)
  • Click "Generate Report"

Reports help track system usage, inventory trends, and departmental needs.
