# ESMIS Backend (API)

This is the Laravel-based backend for the Enhanced Supply Management Information System.

## 🛠 Features
- RESTful API for supply management
- RBAC with Spatie Permissions
- Real-time notifications via Laravel Reverb
- Automated audit logging
- Email notifications for supply requests

## 🚀 Setup
1. `composer install`
2. `php artisan migrate --seed`
3. `php artisan serve`

Refer to the root [README](../../README.md) for full project documentation.
